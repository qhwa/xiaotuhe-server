xiaotuhe-server
===============

Server side sub project of [xiaotuhe.com](http://xiaotuhe.com)

Xiaotuhe is a rapid file sharing server.

## setup

git clone https://github.com/qhwa/xiaotuhe-server.git
cd xiaotuhe-server
bundle install
rails s

then visit [http://localhost:3000](http://localhost:3000)
